🔨 Build APK via GitHub Actions

✅ Cara Build (yang BENAR)

  1. Push game HTML kamu ke folder games/<slug>/ di repo (file utama wajib index.html).
  2. Klik tombol “Build APK via GitHub Actions” di atas.
  3. Klik Run workflow → isi game_folder = slug kamu, app_name = nama aplikasi.
  4. Tunggu ~2 menit (hijau ✅) → scroll bawah → Artifacts → download game-apk → ekstrak → install.
